# Flashcards

Este é um projeto de um sistema de flash cards para ajudar nos estudos.

O usuário pode criar cartões com perguntas e respostas para revisar conteúdos de forma mais fácil e rápida.

## Features

- Criar cartas (flash cards com pergunta e resposta)
- Criar baralhos (organizar cartas por tema)
- Atualizar cartas
- Atualizar baralhos
- Deletar cartas
- Revisar conteúdos de estudo
- Ajudar na memorização

## Armazenamento de Informações

O sistema será um CRUD (criar, ler, atualizar e deletar) feito no terminal.

Não será usado um banco de dados formal. Os dados serão armazenados de forma simulada (mock), apenas para fins de aprendizado e testes.

Tudo será feito via linha de comando, de forma simples e direta.
